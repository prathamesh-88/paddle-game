# Paddle Game

## Summary

Simple paddle game. It is a static website. You can just clone the repo and open the `index.html` file in your browser to play the game.

## How to Play

- Use the up and down arrow keys to move your paddle.
- The objective is to prevent the ball from passing your paddle.
- You can adjust the ball speed and bot speed using the input fields above the game area. This will determine the difficulty level of the game.
- Have fun!
